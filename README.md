## Process Manager GUI

### What is this project?
  * GUI로 프로세스를 관리 할 수 있는 프로젝트 입니다.
### Which OS platforms are supported?
  * 현재는 MacOS(Apple silicon) 만 지원 합니다.
### How can i install it?
```bash
brew tap dev-gyus/gui-process-manager
brew install --cask --no-quarantine gui-process-manager
```
  * --no-quarantine : To allow execution on macOS, the application is installed without quarantine. (MacOS에서의 실행을 위해 격리를 방지하고 설치합니다.)
  * 자세한 내용은 [최신 release 페이지](https://github.com/dev-gyus/gui-process-manager/releases)를 참고해 주세요
